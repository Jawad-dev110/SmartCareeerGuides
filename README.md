🚀 SmartCareerGuides (In Progress)
SmartCareerGuides is a web-based career guidance platform designed to help students discover their ideal career paths based on their interests and personality traits.

🛠️ Current Status: Under Development
This project is currently in the development phase. Core functionalities and system modules are being implemented.

✨ Key Features (Coming Soon)
RIASEC Assessment: A personality-based test to analyze student interests.

AI Career Counselor: An AI-powered assistant for personalized career advice.

University Directory: Information about top universities and programs.

User Dashboard: A personal space for students to save results and manage profiles.

💻 Tech Stack
Backend: Core PHP

Database: MySQL

Frontend: HTML5, CSS3, JavaScript
